# AlgoVoi docs

Source for [docs.algovoi.co.uk](https://docs.algovoi.co.uk).

Built with [Mintlify](https://mintlify.com). Every push to `main` rebuilds and deploys.

## Local preview

```bash
npx mintlify@latest dev
```

Boots a hot-reloading preview at `http://localhost:3000`.

## Structure

- `docs.json` — site config and navigation
- `*.mdx` — top-level pages (introduction, quickstart, support, etc.)
- `concepts/`, `protocols/`, `chains/`, `integrations/`, `api-reference/` — grouped page sections
- `images/` — logos and the favicon

## Reporting issues

For documentation issues, open a PR or an issue on this repo. For platform issues, email [support@algovoi.co.uk](mailto:support@algovoi.co.uk).
