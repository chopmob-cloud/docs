AlgoVoi exists inside an open-protocol ecosystem (x402, MPP, AP2, A2A, Solana Pay, Algorand, VOI, Stellar, Hedera, Base, Tempo). The fastest way to evaluate whether a payment provider is a real participant or a marketing site is to look at what they've contributed back upstream.

Every link below is a public pull request on a public repository. Each one is verifiable from a browser, no AlgoVoi account needed.

## Protocol-level contributions

These land AlgoVoi as a canonical reference inside the protocol's own repository, not just an entry in an external list.

## Why we make this list public

Most payment providers ship a marketing site. The thing that distinguishes a real protocol participant is the public commit history, the PRs that maintainers can review and merge, and the bug fixes that show up in upstream release notes. We're listing all of these in one place so anyone evaluating AlgoVoi (a prospective tenant, a regulator, an investor, a competitor) can verify the contribution footprint without having to take our word for any of it.

If you're a maintainer of an ecosystem repo we should be listed in and aren't, [tell us](mailto:support@algovoi.co.uk).
